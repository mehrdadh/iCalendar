Frequently Asked Questions

How do I import an ICS file to Google Calendar?

Install the iCalendar Chrome extension, click the extension icon, and drag your .ics file onto the popup window. The extension will automatically parse the calendar file and let you add the events to your Google Calendar with one click.

Does this work with VCS files too?

Yes! The iCalendar extension supports both ICS (iCalendar) and VCS (vCalendar) file formats. Simply drag and drop either file type to import calendar events to Google Calendar.

Is the iCalendar Chrome extension free?

Yes, completely free! Download the extension from the Chrome Web Store and start importing calendar files to Google Calendar instantly with no subscription or payment required.

Is my calendar data secure?

Absolutely. All ICS and VCS file processing happens locally in your browser. No calendar data is sent to any third-party servers. The extension only connects directly to Google Calendar using secure OAuth authentication.

Can I choose which Google Calendar to add events to?

Yes! The extension lets you select from all your Google Calendars when importing events. Perfect for organizing work, personal, family, or project-specific calendars.

What information is extracted from ICS files?

The extension automatically extracts event title, start/end times, location, description, attendees, and recurrence rules from ICS and VCS calendar files for seamless import to Google Calendar.
